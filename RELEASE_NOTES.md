# GLASSES VIBE IDE v10.0.4

## 🇬🇧 English
- **Removed Green Status Bar**: The green status bar at the bottom was hiding messages. It has been removed so all chat messages are now fully visible.
- **Bilingual UI**: All interface text is now in both English and Turkish (EN/TR).
- **Chat History Panel**: All your messages and AI responses are now clearly visible in the chat history panel.
- **Improved Layout**: Explorer on the left, Workspace + Chat History on the right.

## 🇹🇷 Türkçe
- **Yeşil Durum Çubuğu Kaldırıldı**: Alt kısımdaki yeşil durum çubuğu mesajları gizliyordu. Tüm sohbet mesajlarının tam görünmesi için kaldırıldı.
- **Çift Dilli Arayüz**: Tüm arayüz metinleri artık hem İngilizce hem Türkçe (EN/TR).
- **Sohbet Geçmişi Paneli**: Tüm mesajlarınız ve AI yanıtları artık sohbet geçmişi panelinde açıkça görünüyor.
- **Geliştirilmiş Düzen**: Sol tarafta Explorer, sağ tarafta Workspace + Sohbet Geçmişi.
