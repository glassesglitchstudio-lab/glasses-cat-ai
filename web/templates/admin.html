📚 API Dokümantasyonu

POST /chat

Normal chat mesajı gönder. AI yanıtı döndürür.

{
  "message": "Merhaba, nasılsın?"
}

POST /voice

Sesli mesaj gönder (speech-to-text sonrası).

{
  "message": "sesli mesaj metni"
}

GET /history

Chat geçmişini getir.

POST /history/clear

Chat geçmişini temizle.

GET /admin/logs

Admin only: Son API isteklerini gör.

Header: X-Admin-Password: glasses_admin_2024

GET /admin/stats

Admin only: İstatistikleri gör.

GET /admin/queue

Admin only: Kuyruk durumunu gör.
